
# 🎓 Student Placement Prediction System

An AI-powered Machine Learning application that predicts a student's placement chances based on academic performance, technical skills, internships, projects, certifications, and communication skills.

## 🚀 Features

- Placement Probability Prediction
- Student Readiness Score
- Skill Analysis Dashboard
- Placement Recommendations
- Strengths & Weakness Analysis
- Interactive Streamlit UI
- Machine Learning-Based Prediction

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Plotly

## 📊 Input Parameters

- Branch
- College Tier
- CGPA
- Backlogs
- Coding Skills
- DSA Score
- Aptitude Score
- Communication Skills
- ML Knowledge
- System Design
- Internships
- Projects
- Certifications
- Hackathons
- Open Source Contributions
- Extracurricular Activities

## 🎯 Output

- Placement Probability (%)
- Placement Status
- Readiness Score
- Skill Analysis
- Recommendations
